This package is a collection of MATLAB code used in the paper

G. Li, Q. Li, S. Li, and W. A. Li,
"On a class of greedy sparse recovery algorithms," Information and Inference: A Journal of the IMA, 2026.

The code in this package is intended to reproduce all the numerical
results reported in the paper. The main scripts for reproducing figures and tables are the files starting with `test_`, including:

test_fig1_varykappa.m
test_fig2_varykappa.m
test_fig3_phasetransition.m
test_fig4_5_6_condQ.m
test_fig7_Run_NewAlgs_4_Noise.m
test_fig8_table3_imags_denoising.m
test_table_1.m
test_table_2.m

Supporting algorithm files, helper functions, images, and saved data
files are also included in this package.

If you use this code in academic work, please cite the paper above.
